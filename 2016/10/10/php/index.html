php

php简单介绍

php是一种后台编程语言，是目前比较主流的后台编程语言。全称是Hupertext Preprocessor,即超文本预处理器。博客系统的核心是由php完成的。facebook也在使用php开发后台。中小型公司也比较偏爱php做后台分开发。

php和前端

php既然是一种后台语言，为什么我们前端也要学习呢。这是为了让我们详细的了解前后台交互理解，更好地明白机制运作的逻辑。在工作中，更方便一个团队的高效运作，前后台对接事宜相互可以更好的沟通。当然，学习php,或许会让你成为一个全栈工程师。O(∩_∩)O~

php的优点

1.开源

2.能运行于各种平台

3.兼容几乎所有的服务器

4.语法简单，容易学习

5.功能强大，能做大型开发

6.能够动态生成web页面内容（混编，php可以嵌入到html代码中）

环境搭建

  • browser
  • Apache
  • PHP
  • MySQL

Apache是web服务器软件，官网地址：http://www.apache.org

MySQL是关系型数据库管理系统 官网地址：http://www.mysql.com

手动配置它们的环境，是相对麻烦的，而XAMPP帮我们集成了这个环境，所以一般推荐使用XAMPP。XAMPP是一个功能强大的软件站集成软件包（Apache+MySQL+PHP+Perl）。

php基础语法

标记

<?php
   //你的代码写在中间
?>

注释

1.多行注释

/*
注释内容
注释内容
*/

2.单行注释

//注释内容

3.文档注释

/**
  *被注释的文档
  *被注释的文档
  */

变量

变量的声明

以$开头，后跟变量名，变量名只能由字母、数字和下划线组成，不以数字开头，区分大小写。不用事先声明数据类型。
例子：

$age = 28;
$color = "red";
$sum = 15;

变量的赋值

1.值赋值：将赋值表达式的值复制给变量（直接赋值）；

2.引用赋值：创建一个变量与另一个变量引用的内容相同

例子：

$a = "hello";
$c = &$a;

变量的变量

$a = "hello";
$$a = "world";
echo $a; //输出 hello
echo $hello; //输出 world
echo ${$a}; //输出 world

超全局变量

php提供了很多预定义的变量，用于提供大量与环境有关的信息

打印/输出超全局变量：prient_r($_SERVER)

1.$_SERVER 服务器变量，该全局变量包含着服务器和客户端配置及当前请求环境的相关信息

$_SERVER['SERVER_NAME']; //当前运行脚本所在的服务器的主机名
$_SERVER['SERVER_ADDR']; //客户端IP地址
$_SERVER['SERVER_URL']; //url的路径部分
$_SERVER['HTTP_USER_AGENT']; //操作系统和浏览器的有关信息

2.$_GET 该变量包含使用GET方法传递的参数的有关信息

url：http://localhost/test.php?id=100&page=2
$id = $_GET['id'];
$page = $_GET['page'];

3.$_POST 该变量包含使用POST方法传递的参数的有关信息；

例子：

html文件中

<form name="reg" action="test.php" method="post">
    用户名：<input type="text" name="username" />
    密 码：<input type="password" name="passwd" />
    <input type="submit" value="提交" />
</form>

php文件中

$username = $_POST['username'];
$password = $_POST['password'];

4.$_REQUEST 该变量记录着通过各种输入方法传递给脚本的变量，如GET POST ，但不要用这个全局变量，因为它不安全而且速度比较慢；

5.$_COOKIE cookie变量数组

6.$_SESSION 会话变量数组

7.$_FILES 与上传文件有关的变量数组

8.$_ENV 环境变量数组

9.$GLOBALS 所有全局变脸数组

常量

常量的定义

常量是指在程序执行中无法修改的值。在脚本执行期间该值不能改变。常量对大小写敏感，通常量名总是大写，常量是全局的，可以在脚本的任何地方引用，常量分为内置常量和自定义常量，使用define()定义。

例子：define(“PI”,3.1415926,false);//第三个参数：是否对大小写不敏感

内置常量

PHP_OS php所在操作系统的名称
PHP_VERSION 当前的php版本号

魔术常量

！注意：这里的下划线是两个英文符号 _ ,请不要写错

__LINE__ ;//文件中的当前行号
__FILE__ ;//文件的完整路径和文件名
__FUNCTION__ ;//函数名称
__CLASS__ ;类的名称
__METHOD__ ;类的方法名

数据类型

标量数据类型

1.字符串

字符串有三种定义方式：单引号，双引号，定界符。单引号字符串中出现的变量不会被变量的值替代。双引号字符串中最重要的一点是其中的变量会被变量值替代。如果遇到美元符号（$）,解析器会尽可能多得取得后面的字符以组成一个合法的变量名，如果想明确地指定名字的结束，用花括号把变量名括起来。

字符串定界的方法使用定界符语法“<<<”，在PHP定界符中的任何特殊字符，都不需要转义，PHP定界符中的PHP变量会被正常的用其值来替换。使用时注意：结束标识符所在的行不能包含任何其他字符，这意味着该标识符不能被缩进，在分号之前、之后都不能有任何空格或制表符。

例子：

$str = <<<A
example of string
using heredoc syntax.
A;

字符串转义：\n 换行 \r 回车 \t 水平制表符（tab键） \ \（反斜杠） \$ $ \” “

2.整型 int

$age = 25;

3.浮点型 float，double

$num = 5.39;

4.布尔型 bool

$bo = TRUE；

$bo = FALSE;

复合数据类型

1.数组

$week = array(‘周一’,’周二’,’周三’);

2.对象

$db = new db;

特殊数据类型

1.资源

$f = fopen(“test.txt”,”r”);

2.null

无，表示没有值，null不表示空格、0。以下情况，则认为是null:没有设置为任何预定义的变量，明确的赋值为null，使用函数unset()清除。

###自动类型转换

因为php对于类型定义的非常松散，所以有时会是根据引用变量的环境，将变量自动转换为最适合对的类型。

类型相关函数

1.gettype() 返回变量的类型

2.is_type() 查看变量是否属于某个类型，是就返回TRUE，否返回FALSE

3.var_dump() 获取变量的值和类型的详细信息
